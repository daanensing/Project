THIS SCATTERPLOT SHOWS THE 30 TOP ARTISTS FOR THE SPECIFIED GENRE ACCORDING TO THE ECHO NEST DATABASE. THE POSITION OF AN ARTIST IN THIS SCATTERPLOT IS BASED ON BOTH ARTIST POPULARITY AND FAMILIARITY. THESE COEFFICIENTS ARE RETRIEVED FROM THE ECHO NEST AND CALCULATED BY MULTIPLE FACTORS LIKE NUMBER OF PLAYS, REVIEWS, BLOGS AND ALIKE. HOVERING OVER AN ARTIST SHOWS THE ARTISTS NAME AND SELECTING AN ARTIST PROVIDES RELATED INFORMATION (LINK TO THE LAST FM ARTIST PAGE, PICTURE, AND SMART RADIO MUSIC PLAYER.

WHEN SELECTING AN ARTIST, A NEW REQUEST TO THE ECHO NEST DATABASE IS MADE WHICH RETURNS 30 RELATED ARTISTS FOR THE SELECTED ARTIST. THESE ARTISTS ARE ALSO SORTED OUT BY POPULARITY AND FAMILIARITY.
